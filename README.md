# enka.discord

enka.discord — бесплатный и простой в использовании Discord-бот для получения ваших билдов/персонажей с enka.network на вашем Discord сервере.

Ссылка на приглашение: https://discord.com/oauth2/authorize?client_id=1296584939583701044

## Возможности

- Получайте билды с аккаунта enka.network или по UID из Genshin Impact и Honkai: Star Rail
- Подключите ваш аккаунт enka.network для лёгкого доступа к вашим билдом
- Использует enka.cards для генерации и отображения карт
- Поиск профиля пользователя на enka.network и получение его имя, аватара и биографии.

## Команды

- `/help` - Получить список команд
- `/build` - Получить ваши билды, если ваш аккаунт подключен
  - `/build uid:<uid>` - Получить билды пользователя по их UID
  - `/build name:<name>` - Получить билды пользователя по их имени
- `/profile` - Получить ваш профиль enka.network, если он подключен
    - `/profile name:<name>` - Получить профиль пользователя по имени
- `/connect name:<name>` - Подключить ваш аккаунт enka.network к вашему Discord аккаунту
- `/disconnect` - Отключить ваш аккаунт enka.network от вашего Discord аккаунта
- `/invite` - Получить ссылку для приглашения бота

## Конфиденциальность

Вся информация, сохранённая в enka.discord, является публичной и может быть легко изменена пользователем.

Единственные данные, которые сохраняются — это аккаунт пользователя enka.network, если он решит его подключить. Мы сохраняем только имя этого аккаунта, но при подключении мы требуем подтверждения, что это действительно вы, через такие способы, как ввод кода в редактируемое поле на профиле.

Другие данные, такие как выбор языка, могут быть добавлены в будущем, но на данный момент сохраняются только подключенные аккаунты enka.network.

## Условия использования

Не используйте этого бота для перегрузки enka.cards или enka.network. Если вы будете пойманы на этом, мы можем удалить ваши разрешения на использование команд бота и ограничить доступ к enka.cards, после чего enka.network может заблокировать вас, если потребуется.

Этот бот предназначен только для демонстрации билдов и просмотра билдов других пользователей. Вы также можете использовать его для просмотра профилей, подключения/отключения аккаунта enka от вашего Discord аккаунта.
